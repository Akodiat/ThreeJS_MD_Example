Example: Three.js for MD data

Double-click on a nucleotide to inspect it.

Exercises

  1. Save a local copy of this repository, either
    • Using git: 
      git clone git@github.com:Akodiat/ThreeJS_MD_Example.git
    • Or by downloading and extracting this zip file
  2. Start the application by running a local static webserver in this directory. You can then make edits to the code and observe the results.
    • If you have python 3, you can type the following command in your terminal: 
      python -m http.server 8080
    • Otherwise, a full list of oneliners is available at https://github.com/imgarylai/awesome-webservers.
  3. Try loading the "aeroplane.csv" file to visualise that instead. Move the point light to cover the whole structure.
  4. Colour nucleotides by strand id, instead of type.
  5. If nucleotide type is "G", set the radius to 0.3. Otherwise the radius should be 0.2
  6. [Advanced] Use python to export or generate your own data (a csv file of 3D points), then load it into this webapp (making any modifications neccesary).
